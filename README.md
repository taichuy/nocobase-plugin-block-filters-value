# @taichuy/plugin-block-filters-value

**English** | [简体中文](./README_CN.md)

Add default value for parent filter block.
Solves the scenario where the parent filter block value cannot be obtained when adding a record after a parent linkage block.

## Introduction

A NocoBase plugin developed by the taichuy team.

Solves the scenario where the parent filter block value cannot be obtained when adding a record after a parent linkage block.

## Usage

1.  **Enable Plugin**: Go to the NocoBase Plugin Manager, search for `@taichuy/plugin-block-filters-value`, and enable it.
2.  **Configure Default Value**: In the default value configuration of a form field, select the variable **Parent filter block record value** (`$tdckParentObject`).
3.  **Select Field**: Supports selecting specific fields of the parent record as the default value.

Documentation: [https://docs.taichuy.com/en-US/nocobase/parent-filter](https://docs.taichuy.com/en-US/nocobase/parent-filter)
