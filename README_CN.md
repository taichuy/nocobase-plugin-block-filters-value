# @taichuy/plugin-block-filters-value

[English](./README.md) | **简体中文**

父级筛选区块添加默认值。
解决场景：在父级链接区块之后，添加时候无法获取父级筛选区块值。

## 介绍：

taichuy团队开发的nocobase插件，

解决场景：在父级链接区块之后，添加时候无法获取父级筛选区块值

## 使用说明

1.  **开启插件**: 进入 NocoBase 插件管理器，搜索 `@taichuy/plugin-block-filters-value` 并启用。
2.  **配置默认值**: 在表单字段的默认值配置中，选择变量，找到 **父筛选区块记录值** (`$tdckParentObject`)。
3.  **选择字段**: 支持选择父级记录的具体字段作为默认值。

详细文档：[https://docs.taichuy.com/nocobase/parent-filter](https://docs.taichuy.com/nocobase/parent-filter)
